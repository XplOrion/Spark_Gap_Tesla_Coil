Dual MOT Spark Gap Tesla Coil (SGTC)

This project is a Tesla coil built using two microwave oven transformers (MOTs) to produce high voltage sparks.

It uses 220 V AC mains power and a push-button switch to turn it on. Three bulbs in parallel are used to limit the current.

The coil has a 14-turn primary coil, a secondary coil with 2500 turns, and 36.5 nf capacitor bank. The resonant frequency is about 156 kHz.

The setup is placed inside a hexagonal wooden case with four cooling fans to remove heat.

Two LEDs are placed near the coil, showing wireless power transfer. A grounded nail placed near the top of the coil helps create visible sparks.

When the push button is pressed, the Tesla coil produces high voltage arcs.
